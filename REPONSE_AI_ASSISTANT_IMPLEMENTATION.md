## 🎯 Comment Utiliser l'Assistant IA

### 1. Accéder à l'Assistant
- Cliquez sur **"Copilot IA"** dans la barre latérale gauche
- Icône : 🤖 (Robot)
- Badge violet "IA"

### 2. Créer une Conversation
- Cliquez sur le bouton **"+ Nouvelle conversation"**
- Une nouvelle conversation vide s'ouvre

### 3. Poser des Questions
Tapez vos questions en français dans le champ de saisie. Exemples :

**🏠 Recherche de Propriétés**
```
"Trouve des appartements 3 pièces à La Marsa"
"Montre-moi les villas avec piscine"
```

**📊 Génération de Rapports**
```
"Résume mes ventes du mois"
"Quelles sont mes statistiques cette semaine ?"
```

**✉️ Rédaction d'Emails**
```
"Écris un email de suivi pour ce prospect"
"Rédige un message de relance professionnel"
```

**💡 Conseils Stratégiques**
```
"Comment négocier avec ce client ?"
"Quelle est la meilleure stratégie pour vendre ce bien ?"
```

**📅 Planning**
```
"Quels sont mes rendez-vous cette semaine ?"
"Organise mon planning de demain"
```

**👥 Gestion des Prospects**
```
"Montre-moi mes prospects actifs"
"Qui sont mes prospects prioritaires ?"
```

### 4. Recevoir les Réponses
- L'IA répond **instantanément**
- Les réponses sont personnalisées selon votre contexte
- Vous pouvez continuer la conversation naturellement

### 5. Gérer les Conversations
- Cliquez sur une conversation dans la liste pour la rouvrir
- Cliquez sur 🗑️ pour supprimer une conversation
- Les conversations sont sauvegardées automatiquement

## 💎 Capacités de l'Assistant IA

### Intelligence Artificielle
L'assistant utilise **GPT-3.5** ou **GPT-4** pour :
- Comprendre vos questions en langage naturel
- Détecter automatiquement votre intention
- Rassembler le contexte pertinent
- Générer des réponses intelligentes et personnalisées

### Détection d'Intention
L'IA détecte automatiquement ce que vous voulez faire :
- **search_properties** : Recherche de biens immobiliers
- **search_prospects** : Recherche de prospects/clients
- **generate_report** : Génération de rapports
- **draft_email** : Rédaction d'emails
- **schedule_planning** : Gestion de planning
- **strategic_advice** : Conseils et stratégies
- **general_query** : Questions générales

### Contexte Intelligent
L'assistant récupère automatiquement :
- Vos propriétés récentes
- Vos prospects actifs
- Vos statistiques de vente
- Vos rendez-vous à venir
- Vos emails récents
- Votre profil utilisateur

## 🔧 Informations Techniques

### Accès
- **URL** : `/ai-assistant`
- **Menu** : "Copilot IA" dans la barre latérale
- **Icône** : 🤖 Bot
- **Badge** : "IA" violet

### API Backend
- **Base** : `/api/ai-chat-assistant/`
- **Endpoints** : 
  - POST `/conversation` - Créer conversation
  - GET `/conversations` - Lister conversations
  - GET `/messages/:id` - Obtenir messages
  - POST `/message/:id` - Envoyer message
  - DELETE `/conversation/:id` - Supprimer conversation

### Authentification
- Requiert un token JWT
- Géré automatiquement par l'interface
- Token stocké dans localStorage

### Base de Données
- **Tables** : `ai_chat_conversations`, `ai_chat_messages`
- **Relations** : Conversations → Messages (1-to-many)
- **Soft Delete** : Oui (deletedAt)

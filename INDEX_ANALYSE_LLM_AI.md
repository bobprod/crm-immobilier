## 🎓 Glossaire Rapide

**LLM** : Large Language Model (modèle de langage IA comme GPT, Claude)

**Router** : Système qui route/dirige les requêtes vers le bon provider IA

**Orchestrator** : Chef d'orchestre qui coordonne les services IA

**Provider** : Fournisseur d'IA (Anthropic, OpenAI, Google, etc.)

**Factory Pattern** : Pattern de conception pour créer des objets dynamiquement

**Token** : Unité de texte (≈ 4 caractères) utilisée pour calculer les coûts IA

**SEO** : Search Engine Optimization (optimisation pour moteurs de recherche)

**ROI** : Return On Investment (retour sur investissement)

**API** : Application Programming Interface (interface de programmation)

**Endpoint** : Point d'accès API (ex: `/llm-config`)

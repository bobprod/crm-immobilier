## 🎯 NOUVEAU SYSTÈME DE QUALIFICATION IA

### **Comment ça Marche?**

1. **Scoring Intelligent**
   ```
   Chaque lead reçoit un score de 0 à 100:

   • Score 75-100 → ✅ QUALIFIÉ (prêt pour contact)
   • Score 50-74  → ⚠️ À VÉRIFIER (validation rapide)
   • Score 0-49   → 🚫 REJETÉ (spam/mauvaise qualité)
   ```

2. **Critères Analysés**
   ```
   Email (30%)      → Syntaxe, domaine, spam
   Téléphone (25%)  → Format, validité
   Nom (20%)        → Complétude, cohérence
   Engagement (15%) → Historique, activité
   Complétude (10%) → Richesse des données
   ```

3. **Interface Simple**
   ```
   ┌────────────────────────────────────┐
   │ 🎯 Qualification Automatique       │
   │                                    │
   │ [50] Total   [32] ✅   [12] ⚠️   [6] 🚫 │
   │                                    │
   │ Filtres: [Tous] [✅] [⚠️] [🚫]    │
   └────────────────────────────────────┘
   ```

## 💡 CONSEILS

### **Pour Maximiser la Qualité**

1. **Commencer par les Qualifiés**
   - Ce sont les meilleurs prospects
   - Prêts pour contact immédiat
   - Taux de conversion le plus élevé

2. **Vérifier Rapidement les "À Vérifier"**
   - Validation rapide nécessaire

   - Déjà identifiés comme spams

## 📱 SUPPORT

### **Questions Fréquentes**

**Q: Puis-je faire confiance au système?**
R: Oui! Le système est calibré pour être précis. Les leads "Qualifiés" (75%+) sont fiables.

**Q: Que faire si un bon lead est rejeté?**
R: Rare, mais possible. Vous pouvez le valider manuellement dans l'onglet "À vérifier".

**Q: Comment améliorer le score d'un lead?**
R: Enrichir ses données: ajouter entreprise, ville, notes, corriger erreurs.

**Q: Les leads rejetés sont supprimés?**
R: Non, marqués comme spam mais conservés. Vous pouvez les récupérer si erreur.

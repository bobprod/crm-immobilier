# Audio Files for Notifications

This directory should contain the following audio files:

- `notification.mp3` - Default notification sound
- `success.mp3` - Success notification sound
- `warning.mp3` - Warning notification sound
- `error.mp3` - Error notification sound

You can use free sound libraries like:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://notificationsounds.com/

For now, placeholder files are created. Replace them with actual audio files.

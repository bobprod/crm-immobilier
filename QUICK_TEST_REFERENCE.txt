```
🔍 IF SOMETHING FAILS
═══════════════════════════════════════════════════════════════════════════════

    STEP 1: Read the Error Message
    ──────────────────────────────
    Example:
    "Timeout: waiting for locator('input#mistralApiKey') to be visible"
    → The input field not found on page


    STEP 2: Check the Screenshot
    ────────────────────────────
    Playwright saves screenshots of failures
    → Look in: test-results/ folder
    → Shows exactly what the browser saw


    STEP 3: Check the Network Tab
    ────────────────────────────
    In Playwright report:
    → Network tab shows API requests
    → If 404 or 500, API has a problem
    → If 200, API worked but frontend didn't


    STEP 4: Use Debug Mode
    ─────────────────────
    Run with debug to see step-by-step:
    $ npx playwright test tests/llm-api-keys-e2e.spec.ts --debug

    This allows you to:
    • Pause execution
    • Inspect elements
    • See what selectors match
    • Execute line by line

🛠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

    Problem: "Port 3000 already in use"
    Solution: $ lsof -i :3000  (Mac/Linux) or Check Task Manager (Windows)
             Kill the process and restart

    Problem: "Database connection failed"
    Solution: Check PostgreSQL is running
             Check DATABASE_URL in .env is correct
             Check database crm_immobilier exists

    Problem: "Tests timeout"
    Solution: Increase timeout in playwright.config.ts
             Check if services are actually running
             Check internet/network connectivity

    Problem: "All 9 fields not visible"
    Solution: Frontend might not have the inputs
             Check: frontend/src/pages/settings/ai-api-keys.tsx
             Look for: input#mistralApiKey, input#grokApiKey, etc.

    Problem: "Success message not appearing"
    Solution: This is OK! The test continues
             Check API response in Network tab
             If status 200, save worked anyway
```

## Quick Command Reference

```bash

# Linux/Mac
./run-playwright-tests.sh

npx playwright show-report

# One test only
npx playwright test -g "should fill and save Mistral API key"

# Verbose output
npx playwright test tests/llm-api-keys-e2e.spec.ts --reporter=list --verbose
```
